# Public Roadmap

This roadmap is intentionally high level. It avoids exposing internal implementation details from the private source repository.

## Current Focus

- production readiness
- public documentation
- cloud deployment planning
- security and trust model
- customer onboarding flow
- portal and connector hardening

## Near-Term Goals

- hosted portal production architecture
- PostgreSQL-backed metadata persistence
- production email confirmation and MFA flows
- secure connector authentication
- signed job manifests
- external queue and worker runtime
- CI/CD with scans and deployment gates
- backup and restore validation

## Product Expansion Goals

- deeper governance reporting
- deeper security posture analytics
- drift and remediation subscription product
- enterprise cost governance and showback
- integration with support and ticketing systems
- enterprise SSO
- customer success and migration playbooks

## Long-Term Vision

VayuNetrix aims to become a guided AWS platform engineering product that helps organizations plan, provision, govern, remediate, and evidence their AWS foundation from one customer-friendly portal.
